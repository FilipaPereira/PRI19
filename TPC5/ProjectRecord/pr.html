Abstract:

Este trabalho de casa tinha como objetivo utilizar a framework Express para NodeJs de modo a construir um servidor para a gestão de alunos. A estrutura de cada aluno é definida por um identificador, um nome, um curso e um array de notas. Cada uma destas, por sua vez, consiste num indicador e no respetivo valor da nota.

O servidor responde a pedidos do tipo GET, POST e DELETE, sendo possível obter a listagem de todos os alunos, adicionar e apagar alunos, bem como consultar toda a informação de cada um deles. É também possível apagar e adicionar notas a cada aluno. Todos os dados dos alunos são guardados num ficheiro json.

Deliverables:

  • Roteador da aplicação
  • Ficheiro pug da página inicial
  • Ficheiro pug da página para visualizar a informação de um aluno
  • Layout de todas as páginas
  • Ficheiro .js auxiliar para apagar alunos e notas
  • Base de dados dos alunos
