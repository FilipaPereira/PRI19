Abstract:

Este TPC tinha como objetivos a criação da stylesheet XSL para transformar o XML do Project Record numa página HTML e a geração de um website para guardar a informação sobre vários arqueossítios.

Este website consiste num índice dos arqueossítios presentes no ficheiro sendo que cada entrada do índice redireciona para a página HTML individual do arqueossítio com a informação correspondente.

Deliverables:

  • Stylesheet XSL para as páginas html dos Arqueossítios
  • XML com informação dos Arqueossítios
  • Índice dos Arqueossítios
  • Stylesheet XSL para o Project Record
  • Project Record - XML
