Abstract:

API para Publicações:

Para este exercício, foi feita a persistência MongoDB, tendo sido criada uma base de dados denominada de pubs com uma coleção com o mesmo nome. Para poder correr a API de dados, é necessário executar o seguinte comando: mongoimport -d pubs -c pubs --jsonArray jcrpubs.json. Depois de gerar o servidor com o Express para a porta 5004, foi criada a rota API que efetua os pedidos GET mencionados no enunciado.

CLAV:

Primeiramente, usei o Postman para efetuar pedidos GET de modo a responder às alíneas propostas, estando a resposta contida no ficheiro alineas.txt. Depois foi desenvolvida a interface, sendo a página inicial uma listagem das tipologias, podendo aceder-se à informação desta carregando no seu ID. Apesar de nas entidades se mostrar clicável, não consegui ligar à página com a informação desta.
