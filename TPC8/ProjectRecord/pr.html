Abstract:

Este trabalho de casa tinha o intuito de criar um servidor em NodeJS para a gestão de uma base de dados de ficheiros em MongoDB. O servidor deve permitir o upload de vários ficheiros, sendo possível fazer o carregamento de múltiplos ficheiros de uma só vez; apresentando também a listagem dos ficheiros presentes na base de dados.

Ao carregar num ficheiro da listagem, são apresentadas as informações referentes ao mesmo através de um modal, dado que se o ficheiro se tratar de uma imagem, esta também é exibida. Os ficheiros/imagens são guardados nas subpastas data e images, dentro da diretoria public.

Deliverables:

  • Modelo de dados (Schema) do Ficheiro
  • Ficheiro com funções de controlo das operações efetuadas na BD
  • Roteador da API
  • Ficheiro pug da página inicial
  • Ficheiro com funções auxiliares em JQuery para o formulário dinâmico
  • Pasta de ficheiros carregados
  • Pasta de imagens carregadas
