Abstract:

Este trabalho de casa tinha como objetivo criar um servidor em NodeJS que recebe pedidos e responde com a página html correspondente. Para tal, a informação presente no Arquivo Musical foi dividida em ficheiros xml separados, cada ficheiro contendo a informação de uma música. Para apresentar a informação de cada uma destas foi também necessário a criação de uma stylesheet, doc2html, para efetuar a conversão dos ficheiros xml para html.

O servidor designado em Javascript responde a pedidos na porta 3132 no seguinte formato: localhost:3132/musicas/12, apresentando a página html do ficheiro doc12.xml que corresponde à música número 12 do arquivo musical.

Deliverables:

  • Servidor
  • Stylesheet para separar o arquivo em documentos individuais
  • XML do arquivo de músicas
  • Stylesheet para gerar a página html de cada música
